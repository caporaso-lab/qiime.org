# qiime.org

Permanent archival landing page served at https://qiime.org.

The original QIIME 1 software was succeeded by [QIIME 2](https://qiime2.org) on 1 January 2018.
This site replaces the legacy multi-page QIIME 1 documentation website with a single landing page that points visitors at:

- [QIIME 2](https://qiime2.org) (active successor project)
- [Zenodo 10.5281/zenodo.18036](https://doi.org/10.5281/zenodo.18036) (QIIME 1.9.1 source release, citable)
- [Software Heritage](https://archive.softwareheritage.org/browse/origin/?origin_url=https://github.com/biocore/qiime) (full `biocore/qiime` git history)
- [Wayback Machine](https://web.archive.org/web/*/qiime.org) (rendered captures of the legacy qiime.org website)

Hosted via GitHub Pages with the `qiime.org` apex domain configured in `CNAME`.
